✅ What is <canvas> in HTML5?
The <canvas> element is used to draw graphics, animations, or games on a web page using JavaScript.

It creates a blank rectangular area where you can draw shapes, lines, images, and text.

By default, it's empty—you must use JavaScript to draw on it.

🧠 Why Use Canvas?
Feature	Description
🎨 Custom Graphics	Draw shapes, lines, and curves
🎮 Games & Animations	Ideal for HTML5-based games
📷 Image Manipulation	Load and edit images
📊 Dynamic Charts	Use libraries (like Chart.js) to show charts
🚀 Fast Performance	Uses pixel-based drawing, good for performance

✅ How to Use <canvas> in HTML5
1️⃣ HTML Part – Define the Canvas Area
html
Copy
Edit

<canvas id="myCanvas" width="300" height="150" style="border:1px solid black;">
  Your browser does not support the canvas element.
</canvas>

2️⃣ JavaScript Part – Draw on Canvas
html
Copy
Edit

<script>
  const canva = document.getElementById("myCanvas");
  const ctx = canva.getContext("2d");

  // Draw a red rectangle
  ctx.fillStyle = "red";
  ctx.fillRect(20, 20, 100, 50);
</script>

✍️ Common Canvas Drawing Methods:
Method	Description
fillRect(x, y, w, h)	Draw filled rectangle
strokeRect(x, y, w, h)	Draw outlined rectangle
clearRect(x, y, w, h)	Clear rectangle area
beginPath()	Start a new shape
arc(x, y, r, sAngle, eAngle)	Draw circle/arc
moveTo(x, y)	Move pen to start position
lineTo(x, y)	Draw line
fillText(text, x, y)	Draw filled text
drawImage(img, x, y)	Draw image

🎨 Difference: SVG vs Canvas
Feature	SVG	Canvas
Type	Vector-based	Pixel-based
Quality	Scales without losing quality	Can blur when scaled
Performance	Slower with many objects	Faster for real-time graphics
Scripting	Not required to draw	JavaScript required
Use Case	Icons, logos, diagrams	Games, animations, real-time charts
